HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML serves as the bones of the webpage as it's responsible for providing structure
  • To make titles use the tag h plus a number 1-6, the larger the number the smaller the title
  • p is used to make a paragraph block
  • ul makes an unordered list
  • ol makes an ordered list
  • li makes a list item
  • img contains info about images that are displayed on the webpage. Can contain various attributes.
  • src defines the location of an image
  • br creates a break line
  • a stands for anchor and creates links
  • href points to the url link

CSS

  • There are three ways to style a page uinsg CSS:
    • Inline CSS: Inline CSS is used for applying attributes to a specific tag. Good for use when needing a quick and permanent change.
    • Internal CSS: Styles an element in the head. That element will then be styled the same throughout the page.
    • External CSS: The most common way to style a webpage. It is kept in a separate file which makes design changes easier.
  • CSS starts with a what you’re wanting to style, called a selector. It is followed by the declaration statements in the declaration block. A declaration is made up of the property, such as color or size, and then the value, such as blue.
  • DRY principle: DRY stands for “Do Not Repeat Yourself”. Instead of writing the same code multiple times, one can use commas to assign code to multiple elements.
  • (More will be added once I can figure out how to show example code)
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Github is used to help teams of developers work on the same project at the same time.
  • Github lets people save iterations of their code so if something goes wrong people can revert back to previous versions without losing progress.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • in git bash, commands relating to git start with git
  • git add -A adds all commits on the currnet branch
  • git commit -m "comment" lets you add a comment to the commit by writing the discriptive comment between the parentheses
  • git pull origin branch-name makes sure the local brach is up to date.
  • git push origin branch-name pushes the code to the remote repository
  • Once the code is pushed to the remote repository, it creates a pull request, or PR.
  • Ideally other peers and or team members will review the PR and make sure it works well. Once all parties agree the PR will be commited to the main code.

JavaScript

  • Java Script was invented by Brendan Eich.
  • JavaScript is dynamic language. Variables in JavaScript are not directly associated with any specific value type and any variable can be assigned values of all type.
  • The set types in JavaScript language consists of primitive values and objects.
  • Primitive values are data that are not an object and have no method or properties
  • Examples of primitive values are:
    • Boolean type: Represents a logical entity and can have 2 values: “True” and “False”.
    • Numbers type
    • String type: Used to represent textual data.
  • Objects are a collection of properties.
  • A key value is either a string value of a symbol value
  • There are two types of data object properties: data property and accessor property
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Conditional Statements and Loops can interrupt the control flow order.
  • If statements are conditional logic.
  • Something is considered ‘Falsy’ if it’s: 0, -0, undefined, NaN, or a boolean data type with a false value.
  • Truthy is all other values
  • If statements are conditional logic used in a block of code dependent on the variable. The computer will go done the list until the condition is met.
    • For example: If our variable is x, we would say “if var is x then y action”.
  • An If statement usually ends with else.
  • An Array is a single variable that holds group data that is typically related.
  • To recall a specific variable from an array we would use the index number. The index number is based on where the variable is in line. Starting with 0 and goes up by one.
  • For loops are a type of loop that uses the predictable pattern of indices to preform a task with all items of an array by allowing a single code block repeatedly.
  • A For loop contains three important statements:
    • The first statement determines the starting point. To do so we declare a variable x and give it the value of 0.
    • Next statement is the condition. For example length. Looking like: x < var length.
    • The final statement allows the array to iterate over each time. For example: x+1”.
      • x++ can serve as a shortcut for writing x+1.
  • A function is a set of instructions that tells a computer how to preform a certain task.
  • Unlike a For loop, a function must be specifically called to function. Kind of like a reference section of a book.
  • The example used was making tea. One must write out the steps to make tea, but one doesn’t want to write it out every time they make a cup of tea. So a function serves to define the steps wanted, and can be called anytime by it’s name and the code will know what to run.
